# BB Kurumsal Tasarım Revizyon 2

Bu sürüm GitHub Desktop üzerinde değişiklik görünmesini garanti edecek şekilde hazırlanmıştır.

## Değişiklikler

- Kurumsal renk sistemi güçlendirildi.
- Sarı vurgu sınıfları mavi/gri kurumsal vurguya çevrildi.
- Header, footer, ana sayfa ve kart vurgularında Revizyon 2 kurumsal çizgi uygulandı.
- Hizmetler sayfası içerik olarak korunacaktır.
- Public klasörüne kontrol amaçlı `bb-kurumsal-revizyon-2.svg` eklendi.

## Güvenlik

`.env`, `.env.local`, token, şifre, API key dosyaları bu pakete dahil edilmemelidir.
