# Bahri Budak Web Sitesi - Revizyon 01

Bu revizyon, bahribudak.com projesini **Örgü / Knitting, Boya / Dyeing ve Apre / Finishing** uzmanlık omurgasına ve **Bahri Budak Kurumsal Kimlik v1.1** standardına bağlayan temel düzenlemedir.

## Kurulum

```bash
npm ci
npm run dev
```

Üretim doğrulaması:

```bash
npm run build
npm run start
```

## Kurumsal Kurallar

- Yalnızca `public/brand/bb-logo-kisa-kurumsal.*` ve `public/brand/bb-logo-tanimli-kurumsal.*` resmî logo olarak kullanılmalıdır.
- Logo parçaları ayrılmamalı, yeniden çizilmemeli ve efekt uygulanmamalıdır.
- Ana yazı ailesi Poppins'tir.
- Ana renkler: `#0B2343`, `#12365E`, `#2EA6D9`, `#5BBBE6`, `#EAF6FC`, `#F5F7FA`, `#4C5561`, `#FFFFFF`.
- Ana uzmanlık mimarisi Örgü, Boya ve Apre'dir.
- Ayrıntılı uygulama sırası `SITE_REVISION_PLAN.md` dosyasındadır.

## Revizyon Durumu

- Faz 1: tamamlandı
