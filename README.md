# Revizyon 36 - Blog Üst Alan Fotoğraf Düzenlemesi

Bu paket blog sayfasındaki boş üst alanı daha profesyonel hale getirmek için hazırlanmıştır.

Yapılanlar:

- Kullanıcının gönderdiği endüstriyel boru / buhar görseli blog üst alanına eklendi.
- Blog üst bölümü iki kolonlu premium yapıya çevrildi.
- Solda başlık ve teknik yayın açıklaması, sağda görsel kart kullanıldı.
- Görsel üzerine kurumsal lacivert overlay eklendi.
- Renkler kurumsal sistemde tutuldu: lacivert, gri, açık zemin, vurgu mavi.
- Renkli emoji ikon eklenmedi.

Commit mesajı:

Blog üst alanına endüstriyel proses görseli eklendi
