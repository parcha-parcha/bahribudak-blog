# Revizyon 35 - Header Logo Ölçüsü ve Hero Geri Alma

Bu paket kullanıcının ölçü tarifine göre hazırlanmıştır.

Yapılanlar:

- Header / küçük yatay logo alanı 9 cm en, 2.5 cm boy mantığına göre ayarlandı.
- Logo bu alan içinde hem yatayda hem dikeyde ortalandı.
- Dikey merkez 1.25 cm mantığıyla orta hizaya alındı.
- Logo etrafında ekstra dış çizgi eklenmedi; logonun kendi beyaz paneli kullanıldı.
- Büyük hero logo kartı bir önceki daha dengeli hale geri alındı.
- 35 / 3 / 01 gibi karışan metrikler yeniden eklenmedi.

Commit mesajı:

Header logo ölçüsü ve hero önceki hale alındı
